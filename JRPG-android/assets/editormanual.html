Quick Overview

When you launch the Map Editor you will be presented with the initial UI which consists of a banner of buttons across the top of the screen and a vertical panel of of buttons.

The top banner contains commands for broad and general map commands.

The smaller pane labeled 'Tile Tools' contains different operations for editing the map.

Main Commands

Exit will exit the map editor and return to the main menu. If you have made changes to the map that have not been saved then you will prompted if you want to save first.

New will create a new blank map at 32x32 tiles dimension.

Open will bring up a list of existing custom maps which you may select to open in order to edit or view.

Save prompts to save the current map to a file.

Undo if there's any previous operation that can be undone this will undo it.

Redo will redo a previously undone operation

Tile Palette brings up a window that displays the available tiles. Click on the image or text in the list to select that tile. The currently selected tile will be displayed in the main commands banner at the far right.

Map Info a dialog will appear allowing you to setup the display name (the name that will appear at map selection) and any mission info you'd like to setup. In addition you also configure the objectives for the map here. Select an objective type from the spinner/combo-box control just under the 'Objectives:' label and click the 'Add' button to the right to add it to the mission.

You can edit or delete objectives by selecting one in the list (when selected it will have a black border surrounding it), and tapping either the edit or delete buttons below. If there is anything you may edit (such as 'KillsRequired' for an 'Annhilate N' objective you will be able to configure it here.

Resize Map will bring up a dialog allowing you to specify dimensions to resize the map to. The default values will match the current map-size, additions / removals based on resizing are made along the top and the right edges.

Gen Rand will bring up a dialog which will allow to configure the random generator to create a new map at at random. This can be useful for finding a layout more or less looks suitable or interesting and then refining it by removing hallways and placing spawns, etc.

Objectives

In general the map editor should be reasonably intuitive to anyone that has worked with a level editing program before or is at least lightly familiar with paint / graphics software.

However, objectives are a game specific item that do need thorough explanation in order to be used correctly without guessing.

Annhilate N is an objective criteria which specifies a number (default of 25) of aliens that the humans must kill for the objective to be marked successful.

Destroy Targets automatically marks generators placed on the map as items that must all be destroyed by the humans in order for the objective to be marked as passed.

Survive For X specifies a number of turns for which the human player must maintain at least 1 surviving marine.

Repair Targets automatically marks generators placed on the map as items that each must be repaired by the human player. Aliens cannot engage the generators unless doubled up with a 'Protect Targets' objective.

Escape the human player must get his pieces off the playing board at exit points on the map (painted in 'Special paint mode') - at least one piece has to make it.

Acquire Relic the humans must acquire an alien relic (placed on map in 'Special paint mode', as long as the human that is carrying the relic is alive this objective will be marked as successful.

Protect Targets the human player must protect the generators placed on the map. As long as at least one generator exists, this objective will be flagged as success.

Multiple Objectives

Objectives run in tandem. If any objective reports 'Failure' then the humans have lost. If all objectives report success, then the humans have 'Won.' While in progress objectives report 'Indeterminate' until a success or failure condition is met.

For more interesting missions you should stack multiple objectives, such as 'Acquire Relic' and 'Escape.' In order to succeed the humans must first obtain the relic and then escape - however if they just escape all their pieces off the board ignoring the relic they will fail when it becomes impossible for them to acquire it without pieces on the board.

Tile Tools

Paint

While paint mode is active tapping a square will paint the currently selected tile from the 'Tile Palette.'

In certain special cases such as doors the painter will automatically place the door as an appropriate overlay item on top of an existingly painted square.

Rotate

Rotate mode will rotate any tile that is tapped by 90 degrees. Use this to orientate tiles in the most aesthetically pleasing fashion or sloppily to give a more varied impression.

Erase

Erase mode results in any tile that is tapped on being erased.

Settings

Settings mode will bring up a dialog for any tile that is tapped on that has special attributes that may be configured.

This is used to add tiles that provide messages to player, configure alien entrance points, or human exit points.

Pieces

Pieces is a special paint mode which will allow you to paint pieces onto the game board.

Primarily you will use this to place the humans that the player has at the beginning of the match.

Special

Special paint mode does a variety of things.

It can place alien extrance points, human exit points, objective items such a generators and edit the settings for a piece that has been placed.

In the case of editing piece settings you can configure what kind of grenades a human is carrying as well as how many. In the case of an alien that you may have placed on the map you can set an activation timer, which will be the number of turns before the AI (ONLY THE AI) will begin to move the piece.

That's most useful for when placing tougher alien types that take more than 1 successful hit to kill which you don't want the swarming ahead right away.

A player controlling the aliens may move this piece as he deems fit as he is expected to understand that it is not as disposable as the lesser aliens that spawn. It is a measure to prevent the AI from doing anything unwanted to soon.
